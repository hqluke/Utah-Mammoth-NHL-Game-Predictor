How We Predict the Winner

Each of the last 8 games is scored between 0 and 1 using a weighted blend of key stats:

Goaltending: 25% Save percentage (normalized .850–.960)
Shooting: 25% Shooting percentage (normalized 5%–20%)
Power Play: 12.5% PP goals scored minus PP goals allowed
Penalty Kill: 12.5% Shorthanded goals scored minus allowed
Faceoffs: 8% Faceoff win % (normalized 35%–65%)
Puck Control: 7% Takeaways minus giveaways
Discipline: 7% Fewer penalty minutes than opponent is better
Physicality: 3% Hits delivered minus hits taken
1

Each game gets a score. The 8 scores are then combined using recency weighting. The most recent game counts for 25% of the total, tapering to 4% for the oldest.

2

That weighted score is blended with zone time data (how much time each team spends in the offensive zone vs. league average), which accounts for 8% of the final performance score.

3

Each team's performance score is combined 50/50 with their win rate over the last 8 games. The result is run through a curve to sharpen separation between teams, and a +4% home ice bonus is applied if Utah is the home team.
